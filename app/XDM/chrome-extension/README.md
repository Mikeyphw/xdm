# XDM Chromium extension

Manifest V3 browser integration for protocol 2.0. Load this directory as an
unpacked extension for development, copy the extension ID shown in the popup,
and use XDM's Browser integration page to install/repair the native host.

Compatible Chromium-family targets include Chrome/Chromium, Edge, Brave,
Vivaldi and Opera where their native-messaging implementation remains Chromium
compatible. Automatic downloads are cancelled only after XDM returns an accepted
queue acknowledgement.
